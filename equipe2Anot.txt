Equipe 2:
Compras e Medicamentos; 
Gestor é responsável por: 
- Cadastrar novos medicamentos
- Aumentar a quantidade de medicamentos já comprados
- Crud nos medicamentos
  - Ler
  - Editar
  - Excluir

Se houver o medicamento já no sistema, verifica a quantidade para a compra.
Compra medicamentos com base nos processos.
Dashboard precisa de:
- Tabela para ver os medicamentos cadastrados
- Tela para cadastrar medicamentos
- Tela para ver os processos pendentes
